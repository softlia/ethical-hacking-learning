## Goal:
Learn to manage files and directories with Linux commands.

## Commands Learned:
| Command | Description | Example |
|----------|--------------|---------|
| `touch` | Create empty files | `touch test.txt` |
| `mkdir` | Create directories | `mkdir projects` |
| `cp` | Copy files | `cp file1.txt file2.txt` |
| `mv` | Move or rename | `mv file1.txt /home/user/` |
| `rm` | Remove files | `rm file1.txt` |
| `rmdir` | Remove empty directories | `rmdir emptydir` |

## Practice Summary:
- Created, copied, moved, and deleted test files.
- Learned that `rm -r` can remove whole folders recursively.
