## Learning Objectives
- Create, copy, move, delete files and directories
- Understand wildcards (*) for batch operations

## Key Concepts
- `rm` deletes files; `rm -r` deletes directories recursively
- `cp` copies files or directories
- `mv` moves/renames files or directories
- Wildcards (`*`) allow batch operations (e.g., *.txt)
- Always be cautious with sudo when deleting files
