# Day 4 — Linux Users & Groups

## Learning Objectives
- Learn how users and groups are structured in Linux
- Understand primary and secondary groups
- Use commands to create, modify, and list users and groups
- Understand sudo privileges and their security implications

## Key Concepts
- /etc/passwd: stores user information
- /etc/shadow: stores hashed passwords
- /etc/group: stores group information
- Every file has a user owner and a group owner
- Groups help manage permissions for multiple users
- `sudo` allows elevated privileges; important for admin tasks
